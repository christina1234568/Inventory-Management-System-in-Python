# Inventory-Management-System-in-Python
Simple inventory management system of PPEs coded in Python.

Collaborated with another partner to make an inventory management system of ppes for hospitals, where we used filed handling to store the inventory, user information, supplier information, distribution information, transaction information.

I was responsible for the initial inventory creation and management, user management, the searching functionalities, that is to search and print the distribution list of any particular item. 

Techniques used were file handling to search, modify, save, and delete information form the .txt files. Try and catch for the initial inventory creation and first user creation. Functions and control flow.
